## 3. 总体结果（COMBINED / 单序列等价）

| 指标 | Baseline | Improved | Delta (Improved-Baseline) |
|---|---:|---:|---:|
| HOTA | 55.921 | 51.992 | -3.929 |
| MOTA | 68.503 | 67.653 | -0.850 |
| IDF1 | 65.519 | 59.890 | -5.629 |
| AssA | 54.380 | 47.703 | -6.677 |
| AssRe | 60.535 | 61.432 | +0.897 |
| IDSW | 230 | 177 | -53 |
| IDs | 320 | 127 | -193 |

## 4. 关键结论

- `IDSW` 降幅：`(230-177)/230 = 23.04%`
- 与你说的“其他地方通常有 20%+ 降幅”一致，这次在 YOLOX_x + SOMPT22-10 上确实达到了。
- 但同时出现明显权衡：`HOTA/IDF1/AssA` 下降，`AssRe` 小幅上升。
