## 9. 简要结论

- 综合 HOTA / RHOTA / IDF1 / AssRe 现在最好的是 `ByteTrack-DualTuned`: `49.426 / 52.533 / 63.000 / 54.035`
- MOTA 与 DetA 最好的是 `HybridSort`: `64.166 / 53.982`
- AssA 最好、IDSW 最少的都是 `BoostTrack`: `49.176 / 2,152`
- 关联速度最快的是 `OcSort`: `91.9 FPS` / `10.89 ms`
- `StrongSort` 在本次同缓存复用对比中最慢，关联阶段约 `282.04 ms/frame`，仅 `3.5 FPS`
- `ByteTrack-DualTuned` 的精度行是后补进旧表的联合调参结果；由于历史 artifact 未保存关联 timing summary，所以该行 `Assoc FPS / Avg ms` 保持为 `--`
