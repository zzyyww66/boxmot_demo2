# SOMPT22 全序列最终结果（2026-02-23）

## 1. 实验口径

- 数据集：`SOMPT22` 全训练序列（9 序列，14100 帧）
- 检测器：`yolov8m_pretrain_crowdhuman.pt`
- 跟踪方法：`bytetrack`
- 评估类别：`pedestrian`（`eval_classes: {1: pedestrian}`）
- 运行方式：先跑原版（从头生成 dets），再跑改进版（复用同一份 dets 缓存）

## 2. 配置文件

- 数据集配置：`boxmot/configs/datasets/SOMPT22-full.yaml`
- 原版配置：`boxmot/configs/trackers/bytetrack_original.yaml`
- 改进版配置：`boxmot/configs/trackers/bytetrack_improved.yaml`
  - 关键参数：`strict_entry_gate=false`

## 3. 运行日志

- 原版：`runs/sompt22_compare_v8m_full_final/baseline_eval.log`
- 改进版：`runs/sompt22_compare_v8m_full_final/improved_eval.log`

## 4. 输出目录

- 原版结果：`runs/sompt22_compare_v8m_full_final/mot/yolov8m_pretrain_crowdhuman_lmbn_n_duke_bytetrack`

## 7. 结论

- 改进版相对原版在身份稳定性上继续有优势：
  - `IDSW: 1117 -> 970`（减少 147）
  - `IDs: 1525 -> 715`（减少 810）
- 同时存在精度权衡：
  - `HOTA` 与 `IDF1` 下降
  - `MOTA` 与 `AssRe` 上升
